CHATGPT DISCORD BOT - MODE B (répond automatiquement partout)

Étapes générales d'utilisation :

1. Installe les dépendances (en local ou sur le serveur d'hébergement) :
   npm install

2. Configure les variables d'environnement :
   DISCORD_TOKEN = le token de ton bot Discord
   OPENAI_KEY    = ta clé API OpenAI

3. Lance le bot :
   npm start

Le bot répondra automatiquement à presque tous les messages envoyés sur ton serveur (sauf les bots et les messages vides).
